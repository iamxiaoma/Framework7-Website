Introduction to Framework7 React

It is highly recommended for you to be familiar with basics of both Framework7 and React.js before getting started.

Framework7 React combines the power and simplicity of React.js with the flexibility and UI of the Framework7 mobile framework to build mobile apps in even more easy and quick way.

Framework7 React helps to keep data and content in more structured way and simplify layout difference for iOS and Material themes (or handle this difference easier).

Framework7 React provides a set of almost all Framework7 elements and components with integration of Framework7 Router to render pages in the "React-way".

This documentation provides reference for Framework7 React components (their usage examples and properties). For their appearance and how do they work it is recommended to check Framework7 Documentation.
